A full-stack furniture e-commerce demo site built with **Django REST API** and **React.js**.

Features
- Product listing from Django backend  
- REST API using Django REST Framework  
- React frontend with Axios  
- Beautiful product card UI  
- Responsive layouts using CSS Grid  

Tech Stack
- Django + Django REST Framework
- React.js (with Axios)
- SQLite Database

How to Run

Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

Frontend
cd frontend
npm install
npm start
